src - zdrojové kódy simulátoru a JUnit testů
lib - externí knihovny (použita knihovna ControlsFX pro select-box umožňující více voleb najednou)

build.xml - ant buildfile

		  - ant compile - kompilace zdrojových kódů
		  - ant compileTest - kompilace JUnit testů
		  - ant test - spuštění JUnit testů
		  - ant dist - vytvoření spustitelného JAR souboru v dist/lib/
